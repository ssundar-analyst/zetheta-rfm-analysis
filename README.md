# Zetheta Customer Segmentation (RFM Analysis)

## Problem Statement
The goal of this project is to segment customers based on purchasing behavior
to identify high-value and at-risk customers.

## Tools Used
- SQL (MySQL)
- Excel
- Power BI

## Dataset
- 50,000+ customer transaction records

## Key Tasks
- Cleaned and standardized date formats
- Calculated Recency, Frequency, Monetary metrics
- Built RFM scores using SQL
- Created Power BI dashboard for visualization

## Insights
- Top 20% customers contributed ~32% of revenue
- Identified 15% churn-risk customers

## Files
- /data → datasets
- /sql → SQL queries
- /dashboard → Power BI files
